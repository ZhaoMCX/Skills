# Agent Framework

Create, adapt, and review project-specific AgentFramework guides.

## Install

Copy this directory to:

``text
%USERPROFILE%\.codex\skills\agent-framework
``

From the source monorepo, install with:

``powershell
.\scripts\install.ps1 -Skill agent-framework
``
