# Unity DOTween

Implement, review, and debug DOTween animation workflows in Unity.

## Install

Copy this directory to:

```text
%USERPROFILE%\.codex\skills\unity-dotween
```

From the source monorepo, install with:

```powershell
.\scripts\install.ps1 -Skill unity-dotween
```

## Contents

- `SKILL.md` - skill trigger, workflow, rules, and examples.
- `references/` - supporting reference material when present.
