# Unity TapTap SDK

Integrate, review, and debug TapTap SDK workflows in Unity.

## Install

Copy this directory to:

```text
%USERPROFILE%\.codex\skills\unity-taptap-sdk
```

From the source monorepo, install with:

```powershell
.\scripts\install.ps1 -Skill unity-taptap-sdk
```

## Contents

- `SKILL.md` - skill trigger, workflow, rules, and examples.
- `references/` - supporting reference material when present.
