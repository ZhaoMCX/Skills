# 官方来源

先读目标项目本地代码和部署配置，再查若依官方仓库和文档。RuoYi-Cloud 的 Boot、Spring Cloud、Alibaba、Nacos、Sentinel、Seata、Vue2/Vue3 组合变化快，服务边界和配置中心内容必须以目标项目为准。

## 官方仓库

- RuoYi-Cloud：https://github.com/yangzongzhuan/RuoYi-Cloud
- RuoYi-Cloud-Vue3：从 RuoYi-Cloud README 或官方账号确认当前维护位置。
- RuoYi-Vue3：https://github.com/yangzongzhuan/RuoYi-Vue3

## 来源优先级

1. 目标项目本地 `pom.xml`、各服务 `bootstrap.yml`、Nacos 配置、SQL、Docker/部署脚本、Feign 接口和 fallback。
2. 对应官方仓库的目标分支、README、提交历史和模板源码。
3. Spring Cloud、Spring Cloud Alibaba、Nacos、Sentinel、Seata、Redis、Gateway、OpenFeign 官方文档。
4. 社区文章、博客和旧快照只能作为经验线索。
