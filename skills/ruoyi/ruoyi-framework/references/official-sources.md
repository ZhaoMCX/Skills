# 官方来源

先读目标项目本地代码，再查若依官方仓库和文档。若依同时维护多个产品族、分支和前端组合，版本、JDK、Spring Boot、`javax`/`jakarta`、Vue2/Vue3、Oracle/fast/Cloud 差异都必须以目标项目为准。

## 官方仓库

- RuoYi 经典单体：https://github.com/yangzongzhuan/RuoYi
- RuoYi-Vue：https://github.com/yangzongzhuan/RuoYi-Vue
- RuoYi-Cloud：https://github.com/yangzongzhuan/RuoYi-Cloud
- RuoYi-Vue3：https://github.com/yangzongzhuan/RuoYi-Vue3
- RuoYi-App：https://github.com/yangzongzhuan/RuoYi-App

## 来源优先级

1. 目标项目本地 `pom.xml`、`package.json`、配置、SQL、生成器模板和已改代码。
2. 对应官方仓库的目标分支、README、提交历史和模板源码。
3. Spring、Vue、Element、Nacos、Seata、DCloud 等第三方官方文档。
4. 社区文章、博客和旧快照只能作为经验线索。
