# 官方来源

先读目标项目本地前端代码，再查若依官方仓库和文档。RuoYi-Vue3 与 RuoYi-Cloud-Vue3 前端模式相近，但接口前缀、网关、部署路径和权限数据来源可能不同。

## 官方仓库

- RuoYi-Vue3：https://github.com/yangzongzhuan/RuoYi-Vue3
- RuoYi-Cloud：https://github.com/yangzongzhuan/RuoYi-Cloud

## 来源优先级

1. 目标项目本地 `package.json`、Vite 配置、`.env.*`、`src/utils/request.js`、路由、Pinia store、API 和页面。
2. 对应官方仓库的目标分支、README、提交历史和模板源码。
3. Vue3、Vite、Element Plus、Pinia、Vue Router、Axios 官方文档。
4. 社区文章、博客和旧快照只能作为经验线索。
