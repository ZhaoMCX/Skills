# Uniapp Development

Work with uni-app, uni-app x, DCloud, HBuilderX, and mini-program targets.

## Install

Copy this directory to:

``text
%USERPROFILE%\.codex\skills\uniapp-development
``

From the source monorepo, install with:

``powershell
.\scripts\install.ps1 -Skill uniapp-development
``
