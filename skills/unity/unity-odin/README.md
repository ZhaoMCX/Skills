# Unity Odin

Use Odin Inspector and Sirenix Serializer workflows in Unity.

## Install

Copy this directory to:

```text
%USERPROFILE%\.codex\skills\unity-odin
```

From the source monorepo, install with:

```powershell
.\scripts\install.ps1 -Skill unity-odin
```

## Contents

- `SKILL.md` - skill trigger, workflow, rules, and examples.
- `agents/` - agent configuration when present.
- `references/` - supporting reference material when present.
