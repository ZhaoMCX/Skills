## Source Policy

- Treat official docs as the source of truth.
- Treat generated community skills, blog posts, and package snippets as non-authoritative.
- For platform-native APIs (`wx.*`, `my.*`, `plus.*`), use official platform documentation and wrap non-cross-platform calls in conditional compilation.
