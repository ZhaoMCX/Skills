# Wechat Miniprogram Devtools

Use official WeChat DevTools CLI and miniprogram-automator workflows.

## Install

Copy this directory to:

``text
%USERPROFILE%\.codex\skills\wechat-miniprogram-devtools
``

From the source monorepo, install with:

``powershell
.\scripts\install.ps1 -Skill wechat-miniprogram-devtools
``
